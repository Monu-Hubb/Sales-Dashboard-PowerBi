Task 4: Sales Dashboard (Power BI)
* Objective
  Create an interactive dashboard to analyze sales performance and support business decision-making.
* Tools
	•	Power BI
	•	Excel
	•	PowerPoint
* Dataset
	•	Superstore Sales Dataset (Kaggle)
	•	999 records
* Key KPIs
	•	Total Sales
	•	Total Orders
	•	Total Customers
* Dashboard Features
	•	Sales trend by month
	•	Sales by category and region
	•	Interactive slicers (Year, Segment, Category)
	•	KPI summary cards
	•	Consistent color theme
* Outcome
  Learned to design interactive business dashboards using Power BI.
